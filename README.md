## • json
&nbsp;

‣ json은 프로그래머가 파일에 정보를 저장하기 위해 만든 방식 중 하나이다. 
(= 파일에 정보를 입력하는 방식)  
‣ nodeJS의 json 이름은 무조건 package.json이다.

• main : 프로젝트의 대표 파일이 무엇인지 알려주는 것  
• scripts : 실행하고싶은 것
• dependencies : 내 프로젝트를 사용하기 위해 필수적으로 있어야 하는 것

## node_modules  
npm으로 설치한 모든 패키지가 저장되는 곳이다.
